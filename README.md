# Electron Text Editor

Десктопний застосунок "Текстовий редактор", розроблений за допомогою **Electron Framework**.  
Програма дозволяє створювати, відкривати, редагувати та зберігати текстові файли.

## Мета проєкту

Ознайомлення з принципами розробки десктопних застосунків із використанням **Electron Framework**, а також реалізація базового текстового редактора з графічним інтерфейсом користувача.

## Функціональність

Застосунок підтримує такі можливості:

- створення нового текстового файлу
- відкриття існуючого файлу
- редагування тексту
- збереження змін
- збереження файлу під новим ім’ям
- перевірка незбережених змін перед закриттям програми

## Технології

У проєкті використано такі технології:

- **Electron Framework**
- **Node.js**
- **HTML**

## Встановлення та запуск

### 1. Встановити Node.js

Завантажити Node.js з офіційного сайту:

https://nodejs.org

### 2. Встановити залежності

У папці проєкту виконати:


npm install


### 3. Запустити застосунок


npm start


Після цього відкриється вікно текстового редактора.

## Меню застосунку

Застосунок має головне меню **Файл**, яке містить:

- **Новий** — створення нового файлу  
- **Відкрити** — відкриття текстового файлу  
- **Зберегти** — збереження змін  
- **Зберегти як** — збереження файлу під новим ім’ям  
- **Вихід** — завершення роботи програми
